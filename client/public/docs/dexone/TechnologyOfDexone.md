# 🚀 Technology Stack of DexOne

This page provides an overview of the technologies powering **DexOne**, CraftDex's upcoming AI-native software development platform.

Our goal is to build a modern, scalable, secure, and cloud-native platform capable of supporting AI-powered development workflows for individuals, startups, and enterprise teams.

> **📝 Note**
>
> **LogChart** was the original internal development codename for this project. The official product name is **DexOne**.

---

# 🏗️ Architecture

DexOne is designed using a modular, service-oriented architecture that emphasizes scalability, maintainability, and performance.

Core architectural principles include:

- Modular architecture
- RESTful & GraphQL APIs
- AI-first workflow integration
- Event-driven processing
- Cloud-native deployment
- Multi-tenant SaaS architecture

---

# 💻 Frontend

The frontend of **DexOne** is built using modern web technologies to deliver a fast, responsive, and intuitive user experience.

- **React** — Build interactive user interfaces
- **Tailwind CSS** — Utility-first CSS framework
- **Vite** — Fast development and optimized builds

---

# ⚙️ Backend

The backend powers APIs, authentication, business logic, and platform services.

- **Node.js** — JavaScript runtime
- **Express.js** — REST API framework
- **MongoDB** — NoSQL database
- **Mongoose** — MongoDB object modeling

---

# 🗄️ Database & Backend Services

DexOne uses modern backend services for authentication, storage, and real-time features.

- **Supabase** — Authentication, Storage, Realtime & PostgreSQL services

---

# 🤖 AI & Automation

Artificial Intelligence is at the core of DexOne, enabling smarter development workflows and automation.

- **Python** — AI workflows and automation
- **OpenAI Models** — Code generation, documentation, and AI assistance
- **Google Gemini Models** — AI-powered reasoning and content generation
- **Custom AI Agents** — Task automation and intelligent workflows

---

# ☁️ Deployment

DexOne is designed to be cloud-native and scalable.

- **Docker** — Containerization
- **GitHub Actions** — Continuous Integration & Deployment (CI/CD)
- **Nginx** — Reverse proxy and load balancing

---

# ☁️ Cloud & Infrastructure

DexOne is powered by modern cloud technologies for reliable, secure, and scalable deployments.

- **Docker** — Containerized application deployment
- **GitHub Actions** — Continuous Integration & Continuous Deployment (CI/CD)
- **Nginx** — Reverse proxy and web server
- **Cloudflare** — CDN, DNS, SSL, and security
- **Kubernetes** *(Planned)* — Container orchestration and scaling
