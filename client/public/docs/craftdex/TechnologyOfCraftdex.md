# 🚀 Technology of CraftDex

CraftDex is powered by a modern and scalable technology stack designed to build high-performance web applications, automation tools, and business solutions. We carefully choose technologies that ensure reliability, security, maintainability, and future scalability.

---

## 🌐 Frontend Technologies

The frontend is built to provide a fast, responsive, and intuitive user experience.

- **React.js** – Component-based UI development
- **JavaScript (ES6+)** – Dynamic client-side functionality
- **HTML5 & CSS3** – Modern web standards
- **Tailwind CSS** – Utility-first responsive styling

---

## ⚙️ Backend Technologies

Our backend architecture is designed for performance, scalability, and secure application development.

- **Node.js** – Server-side JavaScript runtime
- **Express.js** – RESTful API development
- **REST APIs** – Communication between frontend and backend
- **JWT Authentication** – Secure user authentication and authorization
- **Supabase Auth** – User authentication and session management
- **Edge Functions** – Serverless backend logic for lightweight operations

---

## 🗄️ Database

CraftDex uses modern database solutions to manage and scale application data efficiently.

- **MongoDB** – NoSQL database for flexible and scalable data storage
- **Mongoose** – Object Data Modeling (ODM) for MongoDB
- **Supabase (PostgreSQL)** – Managed relational database with real-time capabilities
- **Supabase Storage** – Secure file and media storage
- **Real-time Database** – Live data synchronization for collaborative applications
